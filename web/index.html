Dicas de uso no Moto G35
💡
Resolução recomendada
A tela do Moto G35 é 1600×720. O app usa 1280×720 como padrão — ótimo balanço entre qualidade e performance. Para jogos muito pesados, você pode baixar para 960×540 nas configurações do container.
⚠️
Se um jogo travar
O preset PERFORMANCE é mais agressivo. Se um jogo específico ficar instável ou travar, acesse as configurações daquele container e mude o preset Box64/FEXCore para COMPATIBILITY apenas para aquele jogo.
🔄
Compilação de shaders
Na primeira vez que abrir um jogo, pode haver alguns engasgos enquanto os shaders são compilados. Isso é normal. A partir da segunda vez, os shaders já estarão em cache e o jogo rodará muito mais suave.
Como Gerar o APK — Guia Rápido
🏗️
Build automático — tudo pronto, só conectar o GitHub
O workflow de build já está configurado no projeto
① REPLIT
Conecte o GitHub aqui no Replit (uma vez só)
Na barra lateral esquerda do Replit, procure o ícone de ramificação (parece dois galhos) ou clique nos três pontinhos e procure "Version Control" ou "Connect to GitHub".

Faça login com sua conta GitHub → clique em "Create a GitHub repository" e confirme. O Replit vai enviar todo o código do Gamekt pro para o seu GitHub automaticamente. Não precisa mexer em arquivo nenhum.
② GITHUB
Rode o build e baixe o APK
Abra o repositório criado no seu GitHub → clique na aba Actions → clique em "Build Gamekt pro APK" → clique em "Run workflow" (botão verde) → aguarde ~15 minutos.

Quando aparecer o ✅ verde, clique no build → role até Artifacts → clique em "Gamekt-pro-debug" → baixe o APK → instale no Moto G35!
✅ Resumo: Conecta o GitHub no Replit (uma vez) → vai no GitHub → clica Run workflow → espera 15 min → baixa o APK. Pronto.

Não tem conta GitHub? Crie grátis aqui — só precisa de e-mail, funciona pelo celular.

⚠️ Para instalar no Moto G35: Após baixar o APK, vá em Configurações → Aplicativos → Acesso especial → Instalar apps desconhecidos → habilite o navegador → abra o APK e instale.

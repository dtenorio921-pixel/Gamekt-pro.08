🚀 Enviar Gamekt pro para o GitHub

Cole o seu Token do GitHub abaixo e clique no botão. O sistema vai criar um repositório novo na sua conta e enviar o código automaticamente.

Como criar o Token (30 segundos):
1. Acesse: github.com/settings/tokens/new
2. Em Note escreva: gamekt
3. Em Expiration escolha: 30 days
4. Marque a caixinha: ✅ repo (primeira da lista)
5. Role até o fim e clique: Generate token
6. Copie o token (começa com ghp_) e cole abaixo
